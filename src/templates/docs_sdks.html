SDKs & Libraries

Official and community SDKs for the AI-CBT API.

Python SDK

Official Python library for AI-CBT API.

pip install ai-cbt-api
from ai_cbt_api import Client client = Client(api_key="your_key") result = client.score( student_answer="...", marking_guide="...", question_text="..." )
JavaScript SDK

Official Node.js/Browser library.

npm install ai-cbt-api
const { AiCbtApi } = require('ai-cbt-api'); const client = new AiCbtApi({ apiKey: 'your_key' }); const result = await client.score({ studentAnswer: '...', markingGuide: '...', questionText: '...' });
